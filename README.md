Implementation of `isCyclic` function in TypeScript and Go.

Checks for a cycle in a directed graph.

Implements the algorithm specified [here](https://www.geeksforgeeks.org/detect-cycle-in-a-graph/).

## Run all tests

```sh
make test/all
```

## Run GO tests

```
make test/go
```

## Run TS tests

```
make test/ts
```
